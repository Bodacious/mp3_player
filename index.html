mp3_player by GavinM

A simple mp3 player plugin for rails

To find out more about this plugin please check out this post on Handy Rails Tips: http://handyrailstips.com/tips/7-playing-mp3-s-on-your-rails-site-with-mp3_player

Install

script/plugin install git://github.com/GavinM/mp3_player.git script/generate mp3_player mp3s_folder
